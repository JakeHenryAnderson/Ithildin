# Commercial and Naming

## Viability

| Mode | Assessment |
| --- | --- |
| Portfolio/research | High. |
| OSS | Moderate to high. |
| Commercial | Moderate with a narrow wedge. |

## Plausible Wedge

Self-hosted governed MCP gateway for local/private AI agents.

## Potential Paid Offerings

- hosted control plane;
- enterprise policy packs;
- audit export/compliance templates;
- managed binaries/support;
- regulated SMB support.

## Major Obstacles

- trust burden for endpoint security software;
- enterprise sales cycle;
- liability from unsafe agent actions;
- competing cloud-native offerings;
- solo developer maintenance load;
- need for security review;
- unclear budget owner;
- MCP ecosystem churn.

## Naming

Use Ithildin as the repo/project codename unless trademark review supports commercial use.

Use generic component names in code:

- runtime;
- gateway;
- policy;
- audit;
- approval;
- executor;
- registry.

Add branded names only after the architecture stabilizes.
