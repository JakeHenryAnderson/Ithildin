# Ithildin v0.4 Review Packet

This is the human-readable entry point for the v0.4 review candidate. Generate current command
outputs with:

```sh
make release-check
make v04-review-packet
make review-candidate
```

The generated packet remains under ignored `var/review-packets/`; this document is the stable
committed entry point that explains what to attach and how to interpret it.

## Attachments

- consolidated packet from `var/review-packets/v0.2/GPT-5.5-Pro-consolidated/`;
- `make v04-review-packet` output;
- release-check transcript and release evidence JSON from the latest bundle;
- review-doc hashes and artifact hashes;
- signed-evidence demo summary;
- negative-review transcripts;
- source-review closure matrix.

## Interpretation

v0.4 is a review/evidence handoff. It does not unlock new tool powers and does not represent
production readiness. Use [v0.4-external-review-prompt.md](v0.4-external-review-prompt.md) for the
review request and [v0.4-capability-decision-seed.md](v0.4-capability-decision-seed.md) for the
post-review decision frame.
