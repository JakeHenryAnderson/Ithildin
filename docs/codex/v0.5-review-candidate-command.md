# v0.5 Review Candidate Command

Task 176 adds `make v05-review-candidate`, a higher-level handoff command for the v0.5 review-closure
track.

It runs the existing local review-candidate pipeline, then prepares the v0.5-specific source-review
artifacts:

- review packet source pointers;
- external response intake template validation;
- source-review transcript packet;
- reviewer artifact manifest.

## Command

```bash
make v05-review-candidate
```

The command is intentionally conservative. It does not approve capability expansion, close external
review, add runtime behavior, or add governed tool powers. It exists so the next GPT 5.5 Pro / Very High
or human source-review handoff can be prepared with one command.
