## Deferred Powers

The deferred-power list remains frozen:

- shell execution;
- Docker socket access;
- Kubernetes tools;
- browser automation;
- arbitrary HTTP methods, headers, bodies, cookies, or broad network access;
- broad filesystem writes, deletes, moves, chmod, archive extraction, or plugin-provided writes;
- production identity, OIDC, SAML, SCIM, hosted sessions, or multi-tenant stores;
- runtime Postgres;
- hosted telemetry;
- remote MCP hosting;
- plugin SDK or marketplace;
- secrets-manager tools;
- hosted model serving or LLM proxy;
- external notarization, hosted custody, or official supply-chain signing claims.

## Stop Conditions

Stop implementation and report status if any of these occur:

- a critical/high external finding is open;
- a trust-boundary regression appears in tests;
- a fix requires changing the v0.1 local-preview runtime boundary;
- a new governed tool power appears necessary;
- closure language would imply production readiness, public-preview approval, or completed external
  review before evidence exists;
- the same gate failure repeats three times.
