# Security Claims Freeze

Task 210 freezes security claims for the v0.6 handoff.

Allowed claims:

- local-first governed MCP/tool gateway;
- v0.1 local-preview runtime boundary;
- local tamper-evident evidence;
- narrow built-in tools only.

Required warnings:

- External/source review closure: incomplete.
- Capability expansion: no-go.
- Public/security-product positioning: no-go.
- No new governed tool powers.

Do not claim sandboxing, production identity, immutable audit, compliance-grade evidence, hosted MCP,
runtime Postgres, hosted telemetry, or enterprise security-product readiness.
