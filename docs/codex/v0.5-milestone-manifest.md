# v0.5 Milestone Manifest

This manifest records Tasks 152-180. v0.5 is a source-review closure and capability-decision
preparation wave over the same v0.1 local-preview runtime boundary. It is not a capability-expansion
release.

The machine-readable companion is
[v0.5-milestone-manifest.json](v0.5-milestone-manifest.json). The first task, Task 152, records GPT
5.5 Pro's v0.4 feedback and turns it into the roadmap in
[v0.5-roadmap-from-v0.4-review.md](v0.5-roadmap-from-v0.4-review.md).

## Current State

Tasks 152-179 are complete. Task 180 is planned and must receive a checkpoint commit and gate
output before they are treated as implemented.

## Operating Rule

No task in this manifest may add new governed tool powers, shell execution, Docker socket access,
Kubernetes tools, browser automation, arbitrary HTTP methods/headers/bodies, broad filesystem
writes, production identity, runtime Postgres, hosted telemetry, remote MCP, or plugin SDK work.
