# v0.4 Review Packet Generator

Task 149 adds a v0.4-specific review packet command:

```sh
make v04-review-packet
uv run python scripts/v04_review_packet.py --json
```

This guide is `docs/codex/v0.4-review-packet-generator.md`.

The command reuses the existing release-packet evidence collection and adds v0.4-specific labels,
milestone range, runtime boundary, and review-document pointers. It is intentionally secret-free and
does not run the gate itself.

## Required Pairing

Before external handoff, pair the generated summary with:

- `make release-check`;
- `make review-candidate`;
- `make review-packet-diff-gate OLD=<prior checkpoint> NEW=<current checkpoint>`;
- the generated consolidated packet under `var/review-packets/v0.2/GPT-5.5-Pro-consolidated/`.

## Boundary

The v0.4 packet is a review-candidate summary. It does not close external review, certify
implementation correctness, unlock new tool powers, or change the v0.1 local-preview runtime
boundary.
