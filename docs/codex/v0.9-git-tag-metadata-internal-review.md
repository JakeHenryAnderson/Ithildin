## Reviewed Claims

- The tool uses fixed Git argv over local `refs/tags` only.
- The tool does not use shell execution, remotes, checkout, mutation, arbitrary Git commands,
  caller-controlled argv, or caller-controlled format strings.
- Responses do not include raw tag names, stable tag-name hashes, tag messages, signatures, file
  contents, raw diffs, or raw Git stderr.
- Runtime and policy preview both derive a scoped `git_tags` resource and deny parent-repository
  escape.
- Lightweight tags, annotated tags, non-commit targets, unsupported selector fields, malformed
  selectors, limit truncation, and casefold ambiguity are covered by implementation behavior or
  focused tests.
- MCP exposure routes through the governed tool-call path.
- Audit metadata is limited to selector kind, counts, truncation, skipped-count, and output-policy
  booleans.
