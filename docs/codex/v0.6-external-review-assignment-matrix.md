# v0.6 External Review Assignment Matrix

This matrix turns the external-pending rows in the source-review closure matrix into planned review
work. It does not close any row. A row may move from `not started` only after a real external review
response is normalized, assigned an access level, ingested as structured findings or explicit
no-finding evidence, and linked back to the closure matrix.

## Review Rules

- Source-level review is required for implementation-sensitive surfaces.
- packet-only review may comment on documentation, process, or evidence clarity, but it cannot close
  a source-level claim unless the closure matrix explicitly says packet-only is sufficient.
- Internal AI/subagent review is useful pressure testing, but it cannot close external rows.
- Critical/high findings stop capability expansion and pause unrelated work until fixed or formally
  accepted-deferred by a boundary decision.
- No row in this matrix approves new governed tool powers.

## Expected Output Convention

- Raw review output: `var/review-runs/v0.6/<review-id>/raw-response.md`
- Normalized review output: `var/review-runs/v0.6/<review-id>/normalized-response.json`
- Finding namespace: `EXT-###`
- Finding records, after intake: `docs/codex/findings/ext-###-<slug>.md`
